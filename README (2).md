# ملفات المشروع كاملة

هذه السبيس مخصصة لحفظ **ملفات مشروع الحجز بالكامل** مع شرح واضح لكل جزء.

## ما الموجود داخل السبيس
- كود تطبيق Android (Kotlin): داخل مجلد `app/`
- سكربت Python: ملف `fixed_script.py`
- سيرفر الداش بورد (FastAPI): داخل `control_server/`
- ملفات شرح تفصيلية (عربي): داخل `docs/`

## ملفات الشرح المهمة
- `docs/CODEBASE_EXPLAINED_AR.md` شرح هيكل الكود
- `docs/SCRIPT_DETAILS_AR.md` شرح السكربت بالتفصيل
- `docs/DASHBOARD_GUIDE_AR.md` شرح الداش بورد وكيفية الاستخدام
- `docs/CODE_PROTECTION_GUIDE_AR.md` شرح حماية الكود ومنع سرقته

## ملاحظة
هذه السبيس هدفها الأساسي **أرشفة وعرض ملفات المشروع** مع الشرح، وليس تشغيل تطبيق Android داخل Hugging Face.

## الرابط المباشر للعرض
- الصفحة الرئيسية: `index.html`
