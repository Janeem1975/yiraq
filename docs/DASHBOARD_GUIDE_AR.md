# شرح الداشبورد (داخل التطبيق)

هذا الملف يشرح طريقة استخدام الداشبورد بعد التحديثات.

## التبويب 1: الحسابات

### أ) الإعدادات العامة
- إدخال `AppCheck JWT` العام للتطبيق.
- إدخال بيانات البروكسي العام (اختياري).
- زر `حفظ الإعدادات`.
- زر `تعميم AppCheck`: ينسخ AppCheck العام على كل الحسابات كإعداد يدوي.
- زر `تعميم Proxy`: ينسخ Proxy العام على كل الحسابات كإعداد يدوي.

### ب) إضافة/تعديل حساب
1. رقم الهاتف + OTP.
2. إدخال بيانات المواطن.
3. اختيار الدائرة.
4. (اختياري) إعدادات شبكة خاصة بالحساب:
   - AppCheck خاص بالحساب.
   - Proxy خاص بالحساب.
5. حفظ الحساب.

### ج) الحسابات المحفوظة
- تعرض كل حساب مع:
  - رقم الهاتف.
  - الاسم.
  - الدائرة.
  - هل AppCheck عام أم مخصص.
  - وضع الاتصال الفعلي للبروكسي.
- أزرار:
  - تحميل للتعديل.
  - حذف.

## التبويب 2: التشغيل

### أ) إعدادات التشغيل
- تشغيل لحساب واحد أو لكل الحسابات.
- اختيار التاريخ عبر الشرائح + كتابة يدوية (3 سلايدرز).
- أي تاريخ قديم يُعدّل تلقائيًا إلى الغد.
- اختيار باقة الصباح/المساء.
- اختيار وقت واحد أو كل أوقات الباقة.

### ب) التحكمات الرئيسية (بالألوان)
- **تشغيل (أخضر)**: بدء/استئناف التشغيل.
- **Pause (أزرق)**: إيقاف مؤقت.
- **إيقاف (أحمر)**: إيقاف نهائي للحسابات المحددة.

### ج) التحكم لكل حساب
- في وضع "كل الحسابات" يظهر تحكم مباشر لكل حساب:
  - تشغيل/Pause/إيقاف بشكل مستقل.

### د) سجل التنفيذ
- عرض اللوج لحظيًا.
- نسخ اللوج بالكامل.
- نسخ آخر مشكلة.

## التبويب 3: المحجوزات
- يعرض تفاصيل الحجوزات الناجحة:
  - الإيميل.
  - الهاتف.
  - الاسم.
  - الدائرة.
  - التاريخ/الوقت.
  - وقت التسجيل.
- زر نسخ كل النتائج.

## ملاحظات مهمة
- لا يتم كسر منطق التشغيل الأساسي؛ تم إضافة طبقة تحكم فقط.
- عند وجود خطأ إعدادات (AppCheck/Proxy) يظهر السبب بوضوح قبل أو أثناء التشغيل.
